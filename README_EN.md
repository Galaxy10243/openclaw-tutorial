## 2. Installation

### Requirements

- Node.js 18+
- Python 3.10+
- Windows / macOS / Linux

### Steps

```bash
# 1. Install OpenClaw
npm install -g openclaw

# 2. Initialize
openclaw init

# 3. Start gateway
openclaw gateway start
```

## 4. Configure Telegram

### 4.1 Create Bot

1. Open Telegram, search @BotFather
2. Send `/newbot` to create bot
3. Copy bot Token

### 4.2 Configure OpenClaw

```bash
openclaw config set channels.telegram.accounts.default.botToken "YOUR_TOKEN"
```

### 4.3 Restart

```bash
openclaw gateway restart
```

## 6. Install Skills

